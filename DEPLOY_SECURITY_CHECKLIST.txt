문서 마스킹 도구 배포 전 보안 체크리스트

1) 민감 데이터 포함 여부
[ ] 테스트/검증 JSON(cross_validation*, official_doc_validation*, regression_cases*) 배포 제외
[ ] 사용자 실문서(.pdf/.txt/.doc 변환본) 배포 폴더 포함 금지
[ ] 실행 로그/리포트(*.report.*.json, *.masked.*, *.extracted.*) 배포 제외

2) 비밀정보/키 점검
[ ] .env, .pem, .key, 토큰 문자열 포함 파일 배포 제외
[ ] 코드 내 API Key/Password/Token 하드코딩 없음 확인

3) 오프라인 동작 보장
[ ] 기본 동작에서 외부 API 호출 없음
[ ] 네트워크 차단 환경에서 실행 테스트(가능한 환경에서)

4) 배포 패키지 최소화
[ ] tests/, __pycache__/, 개발 스크립트 제외
[ ] 빌드 산출물은 최종 실행 파일 + 필수 런타임만 포함

5) 운영 정책
[ ] 출력 파일 저장 위치 안내(보안 폴더 권장)
[ ] 보관기간/삭제정책 안내(예: N일 후 수동 삭제)
[ ] 관리자용 배포 해시(SHA256) 생성 및 전달
